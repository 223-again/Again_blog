# 星野の栖息地

一个记录随笔与代码技术的个人博客。毛玻璃（Glassmorphism）风格，单文件、纯静态、即开即用。

## 特性

- 🪟 毛玻璃风格 UI，动态渐变背景 + 萤火虫粒子
- 🌗 深色 / 浅色模式，自动记忆偏好
- 🔍 文章实时搜索与分类筛选（随笔 / 技术 / 生活）
- 📖 沉浸式阅读视图，带阅读进度条
- 📱 响应式设计，适配手机 / 平板 / 桌面
- ⚙️ 顶部集中配置区，改站名、头像、简介只需改一处

## 本地预览

直接用浏览器打开 `index.html` 即可，无需任何依赖或构建。

## 自定义

打开 `index.html`，找到脚本顶部的 `CONFIG` 配置区，修改站名、昵称、关于我、社交链接等即可。文章内容在同一脚本的 `posts` 数组里。

## 部署

已托管于 GitHub Pages。推送到 `main` 分支后自动更新。
